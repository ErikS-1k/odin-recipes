# odin-recipes

Una vez terminado el proyecto me voy a demostrar que aprendi todo sobre:
- HTML boilerplate structure
- Headings (h1-h6)
- Parrafos
- Strong y em
- Links (absolutos y relativos)
- Imágenes
- Listas (ordered y unordered)
- Comentarios en HTML
- Estructura de carpetas y rutas relativas (../)
